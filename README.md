# IT Sector Salary & Remote Work Analysis
This Power BI dashboard provides a comprehensive analysis of the IT labor market. By evaluating 54 job roles, the report identifies key trends in compensation, company size distribution, and the global shift toward remote work.

# Key Insights
Compensation Leaders: Machine Learning Engineers command the highest average salary at 107.43K.

Experience Gap: Entry-level salaries average 60K, while market capacity reaches a maximum of 250K.

Organizational Balance: Salary distribution is nearly equal across Large (35.4%), Medium (32.1%), and Small (32.4%) companies.

Remote Work Adoption: The tech industry shows a high remote work ratio, currently averaging 69.26 globally.

# Technical Features
Interactive Slicers: Users can filter data by Experience Level, Job Role, and Company Location.

Global Mapping: Visualized remote_ratio across international borders using Microsoft Bing integration.

KPI Tracking: Real-time metrics for Total Jobs, Average Salary, and Salary Range.
